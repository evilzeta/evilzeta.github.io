Linux Shell 编程：从入门到提高

一、Shell 编程基础

  1. 什么是 Shell？

    • Shell 是用户与 Linux 内核交互的命令行解释器。
    • 常见的 Shell：Bash（默认）、Zsh、Fish等。
    • Shell 脚本：以 .sh 结尾的文本文件，包含一系列命令。

  2. 第一个 Shell 脚本

    • #!/bin/bash（Shebang）：指定脚本解释器。
    • echo：输出内容到终端。

  3. 变量与输入输出

    • 变量定义：变量名=值（注意等号两边不能有空格）。
    • 输入命令：read
